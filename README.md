# Portfolio_ScTec
Projeto "Desafio Prático" do curso introdutório "Introdução à Programação Front-End e Back-End - ScTec"



## Sobre o projeto
Esse portfólio foi feito para o desafio prático do ScTec.
Ele é simples e funcional


## Tecnologias utilizadas
HTML - Estrutura da página  
CSS - Estilização e layout  
JavaScript - Interações e efeitos visuais  


## Funcionalidades
- Navegação por atalhos no menu
- Seção "Sobre mim"
- Cards de habilidades técnicas
- Cards de projetos com link para o código
- Efeitos interativos ao passar o mouse
- Seção de contatos com acesso ao GitHub e e-mail


## Estrutura do projeto
- index.html - Estrutura  
- style.css - Estilização  
- script.js - Interações  


## O que eu aprendi nesse projeto
- Organização de layout com Flexbox e Grid
- Manipulação do DOM
- Uso de `addEventListener`
- Importância de manter o código organizado
